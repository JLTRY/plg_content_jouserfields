# plg_content_jouserfields 1.0.2
## Quick Start

Download <a href="https://github.com/JLTRY/plg_content_jouserfields/releases/latest" target="_blank">latest version</a> of plugin

## Requirements

Joomla 5.0+ 

## Features
Joomla plugin allows to insert user fields in articles

Syntax: insert ```{userfields name=field_name}``` in an article

This will insert the field field_name associated to the current user in the article
